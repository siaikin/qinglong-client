<Callout type="warn">
  此工具会直连你的青龙面板执行真实 API 调用。开发环境下，请求经本站 `/qinglong-proxy` 转发至下方填写的真实面板地址（含 `/ql` 前缀时照常填写），可绕过浏览器 CORS。生产环境（静态部署）无代理，需面板允许跨域或通过反向代理同域部署。请确保应用已授权对应 scope；部分接口会产生副作用。
</Callout>
